# sprig-security-rbac
Demonstration of role based authentication and authorization using spring security and jwt


### Technology used
- Java 17
- Spring boot 2.7.2
- Jwt
- MySQL

### Features
- CRUD user
- CRUD role
- Assign/Revoke role
- Extend role with special privileges
- Revoke special privileges


### Default root user
```
email: admin@test.com
password: admin
role: admin 
privileges: all
```
